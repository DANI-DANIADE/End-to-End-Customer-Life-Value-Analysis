# End-to-End-Customer-Lifetime-Value-Analysis
Analyze CLV to identify high value customers and guide decisions on retention, marketing spend, and customer acquisition strategy.

# Dataset
This project uses a synthetic dataset designed to mimic real-world customer transaction behavior, including purchase patterns, customer segmentation, and revenue distribution.

# Tools
- Excel (Data Cleaning)
- SQL (Data Analysis)
- Tableau (Visualization)

# Data Process
1. Cleaned raw data using Excel (handling missing values,duplicates and formatting)
2. Performed analysis using SQL (CLV, RFM segmentation, churn analysis)
3. Visualized insights using Tableau dashboard

# Key Insights
- Customers in the "At Risk" segment contribute the highest revenue, indicating a high dependency on potentially churning customers  
- Direct channel generates the highest CLV, significantly outperforming other acquisition channels  
- Revenue is heavily concentrated in a few categories, with Electronics leading by a large margin
